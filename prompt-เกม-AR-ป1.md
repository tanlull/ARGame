# Prompt: เกม AR สำหรับเด็ก ป.1 (จับคู่คำศัพท์ + บวกเลข)

> นำ prompt ด้านล่างนี้ไปวางใน Claude / ChatGPT / AI ที่เขียนโค้ดได้ เพื่อให้สร้างเกมออกมาเป็นไฟล์ `index.html` ไฟล์เดียว
> (ผมได้สร้างเกมตัวอย่างตาม prompt นี้ให้แล้วในไฟล์ `index.html`)

---

ช่วยสร้างเกม AR แบบไฟล์เดียว `index.html` โดยใช้ HTML, Tailwind CSS และ JavaScript

**กลุ่มเป้าหมาย:** นักเรียนชั้นประถมศึกษาปีที่ 1 (อายุ 6–7 ปี) — ใช้คำสั่งและคำอธิบายเป็นภาษาไทยทั้งหมด ตัวอักษรใหญ่ อ่านง่าย สีสันสดใส

**หัวข้อเกม:** **เกมจับคู่มหัศจรรย์ AR: คำศัพท์และบวกเลข ป.1**

## แนวคิดเกม
ผู้เล่นใช้มือผ่านกล้องเว็บแคม หรือใช้เมาส์/สัมผัส (touch) เพื่อ "หยิบ" การ์ดคำตอบที่ลอยอยู่ แล้วนำไปวางบน "ช่องโจทย์" ที่ถูกต้อง เช่น หยิบการ์ดคำว่า `CAT` ไปวางบนช่องที่มีรูป 🐱 ถ้าถูกการ์ดจะล็อกเข้าช่องและได้คะแนน

## เทคโนโลยีที่ต้องใช้
* **MediaPipe Hands** สำหรับตรวจจับมือผ่านเว็บแคม
* **Canvas** ซ้อนทับวิดีโอ (วาดการ์ด ช่องวาง เคอร์เซอร์ และโครงมือ)
* **Pinch gesture** (นิ้วโป้งแตะนิ้วชี้) สำหรับหยิบ/ปล่อยการ์ด
* **Mouse/Touch fallback** — ถ้ากล้องหรือ AI ใช้ไม่ได้ ต้องเล่นด้วยเมาส์หรือนิ้วบนจอสัมผัสได้
* ใช้ `requestAnimationFrame` สำหรับ draw loop
* ภาพจากกล้องให้แสดงแบบกระจกเงา (mirror) เพื่อให้เด็กขยับมือได้ตรงธรรมชาติ

## รูปแบบหน้าจอ
* **หน้าเริ่มต้น** — ชื่อเกม + ปุ่ม "เข้าสู่เกม"
* **หน้าคำแนะนำ** — บอกวิธีเล่นด้วยภาพและข้อความสั้น ๆ + ปุ่ม "เริ่มเล่น!"
* **หน้าหลัก** — มีการ์ดคำตอบลอยอยู่ด้านบน และช่องโจทย์ 4 ช่องเรียงด้านล่าง ผู้เล่นลากการ์ดมาจับคู่

## ข้อมูลด่าน (3 ด่าน ด่านละ 4 คู่)
**ด่าน 1 — คำศัพท์ภาษาอังกฤษ (สัตว์):** จับคู่รูปสัตว์กับคำศัพท์อังกฤษ
* 🐱 = CAT, 🐶 = DOG, 🐘 = ELEPHANT, 🐟 = FISH

**ด่าน 2 — การบวกเลข (ผลบวกไม่เกิน 20):** จับคู่โจทย์กับคำตอบ
* 1 + 2 = 3, 4 + 3 = 7, 5 + 5 = 10, 6 + 2 = 8

**ด่าน 3 — คำศัพท์ภาษาไทย:** จับคู่รูปกับคำศัพท์ภาษาไทย
* 🐘 = ช้าง, 🍎 = แอปเปิล, ☀️ = พระอาทิตย์, 🏠 = บ้าน

> หมายเหตุ: ออกแบบโครงสร้างข้อมูลด่านให้แก้ไข/เพิ่มคำศัพท์และโจทย์ได้ง่าย (เก็บเป็น array) เพื่อให้ครูปรับเนื้อหาเองได้

## ระบบคะแนน
* แต่ละด่าน 100 คะแนน (จับคู่ถูก 1 คู่ = 25 คะแนน เพราะมี 4 คู่)
* จับคู่ผิด **ไม่หักคะแนน** การ์ดเด้งกลับที่เดิม และขึ้นข้อความให้กำลังใจ "ลองอีกครั้งนะ"
* เล่นครบ 3 ด่าน แสดงข้อความ "🎉 สำเร็จ! เก่งมาก!"
* แสดงคะแนนรวม (เต็ม 300) และปุ่ม "เล่นอีกครั้ง"

## UI / การออกแบบ
* สไตล์ **Glassmorphism** (พื้นผิวกระจกฝ้า โปร่งแสง เบลอ)
* การ์ดใหญ่ ตัวอักษรใหญ่ อ่านง่าย สีสด ตัดกันชัด เหมาะกับเด็กเล็ก
* ใช้ฟอนต์ไทยที่อ่านง่ายและน่ารัก (เช่น Google Font "Mali")
* มีเสียงตอบสนองสั้น ๆ เมื่อจับคู่ถูก (เสียงดี๊ง) และผิด (เสียงตุ๊บเบา ๆ)
* แสดงสถานะเคอร์เซอร์เป็น 4 สี:
  * 🔵 ฟ้า = ปกติ
  * 🟡 เหลือง = ชี้โดนการ์ด (หยิบได้)
  * 🔴 แดง = ปล่อยการ์ดผิดช่อง
  * 🟢 เขียว = กำลังถือการ์ด / จับคู่สำเร็จ
* มีแถบบอก ด่านที่เล่น คะแนน และจำนวนคู่ที่จับถูกแล้ว

## ข้อกำหนดทางเทคนิค
* เขียนทุกอย่างในไฟล์ `index.html` ไฟล์เดียว (โหลด Tailwind และ MediaPipe จาก CDN ได้)
* โค้ดต้องรันผ่าน local server ได้ เช่น VS Code Live Server (กล้องต้องใช้ผ่าน `http://localhost` หรือ `https`)
* ถ้า MediaPipe โหลดไม่สำเร็จ หรือผู้ใช้ไม่อนุญาตกล้อง ต้องสลับไปโหมดเมาส์/สัมผัสอัตโนมัติ และยังเล่นจบเกมได้
* มีปุ่มสลับโหมด "ใช้กล้อง / ใช้เมาส์" ให้ครูเลือกได้เอง
* รองรับทั้งจอคอมพิวเตอร์และแท็บเล็ต (responsive)
